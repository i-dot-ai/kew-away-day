Build an interactive Cabinet Secretary objectives dashboard as a single HTML file.

Use the objectives data from `data/OBJECTIVES.md` as the data source. The dashboard should:

- Display all objectives and their associated actions in a clear, visual layout
- Use the GOV.UK Design System colour palette and styling as described in CLAUDE.md, enhance it to show the cab sec's name and make it black themed
- Use the data available within the data/ directory and don't make up anything.
- Be interactive — let the user explore, filter, or navigate between objectives
- Look professional and polished, suitable for a senior leadership audience
- Open automatically in the browser when complete

Put the project in `projects/cab-sec-dashboard/`.
